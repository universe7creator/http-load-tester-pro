Frequently Asked Questions

What's the maximum number of requests I can test?
The Pro version supports up to 1000 concurrent requests per test with a maximum of 1000 total requests. This is more than enough for most API load testing scenarios.
Can I test APIs that require authentication?
Yes! The Pro version supports custom headers, so you can include Authorization tokens, API keys, or any other headers your API requires.
What metrics are included in the results?
You get comprehensive metrics including total requests, successful/failed counts, requests per second, total time, and detailed timing breakdowns (min, max, average, P50, P95, P99 percentiles).
Is this a subscription or one-time purchase?
HTTP Load Tester Pro is a one-time purchase of $19. You get lifetime access to the tool and all future updates at no additional cost.
Can I export my test results?
Yes! Pro users can export test results as JSON for further analysis, reporting, or integration with CI/CD pipelines and monitoring tools.
